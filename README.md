# Eco Revision — Spotlight 7 Module 8

Interactive mini-site for vocabulary and grammar revision.

## Features
- vocabulary cards
- front side: word only
- back side: local image + translation + transcription
- sound button
- light / dark theme
- context tasks
- grammar by topics
- final test
- achievements / badges
- adaptive design for phones, tablets and PC
